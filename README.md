# Web App d’Affectation d’Étudiants (P3 – ESIEE PARIS)

## Objectif du projet
Ce projet a pour but de concevoir une application web permettant de gérer et d’optimiser l’affectation d’étudiants à différents contextes :  
- Projets académiques  
- Groupes de travail  
- Mobilités internationales  

L’application vise à automatiser le processus en prenant en compte diverses contraintes et préférences.

---

## Fonctionnalités principales
- Gestion des affectations d’étudiants selon :
  - Les vœux et préférences (projets, coéquipiers, etc.)
  - Les critères spécifiques (classement, langue, capacités, etc.)
  - Les tailles minimales et maximales des groupes  
- Interface web interactive pour la visualisation et la gestion des groupes  
- Système d’authentification et gestion des utilisateurs  

---

## Défis du projet
- **Défi algorithmique :** mise en place d’un algorithme génétique pour l’affectation sous contraintes.  
- **Défi technique :** développement d’une application web ergonomique et moderne.
